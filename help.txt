# 📘 ΕΓΧΕΙΡΙΔΙΟ ΒΟΗΘΕΙΑΣ - ΠΡΟΣΟΜΟΙΩΤΗΣ ΔΙΚΤΥΩΝ
# ==============================================

## 🎯 ΒΑΣΙΚΕΣ ΛΕΙΤΟΥΡΓΙΕΣ

### 📍 ΠΡΟΣΘΗΚΗ ΣΥΣΚΕΥΩΝ
• [ΚΛΙΚ]: Κάντε κλικ σε συσκευή από το αριστερό πάνελ
• [ΠΛΗΚΤΡΟΛΟΓΙΟ]: Ctrl + A (προσθήκη νέας συσκευής)
• [ΜΕΤΑΚΙΝΗΣΗ]: Σύρετε συσκευές στο workspace

### 🔗 ΣΥΝΔΕΣΗ ΣΥΣΚΕΥΩΝ
1. Κάντε κλικ στο κουμπί "Σύνδεση" 🔗
2. Επιλέξτε την πρώτη συσκευή
3. Επιλέξτε τη δεύτερη συσκευή
4. Οι συσκευές θα συνδεθούν αυτόματα

### ⚙️ ΡΥΘΜΙΣΕΙΣ ΣΥΣΚΕΥΩΝ
• [ΚΛΙΚ]: Σε συσκευή για πάνελ ρυθμίσεων
• [ΡΥΘΜΙΣΕΙΣ]:
  - Διεύθυνση IP
  - Subnet Mask
  - Gateway
  - DNS Server
  - Domain Name (για servers)

## 🔧 ΕΡΓΑΛΕΙΑ ΔΙΚΤΥΟΥ

### 🚀 ΔΟΚΙΜΗ PING
1. Κάντε κλικ σε μια συσκευή
2. Πατήστε "Δοκιμή Ping" στο πάνελ
3. Επιλέξτε προορισμό
4. Το σύστημα θα προσομοιώσει ping

### 🛣️ ΔΟΚΙΜΗ ΔΙΑΔΡΟΜΗΣ
1. Πατήστε "Διαδρομή" 🛣️
2. Επιλέξτε πρώτη συσκευή
3. Επιλέξτε δεύτερη συσκευή
4. Το σύστημα θα εμφανίσει τη διαδρομή

### 🌐 DNS RESOLUTION
• [ΑΥΤΟΜΑΤΟ]: Χρησιμοποιεί τον ρυθμισμένο DNS
• [ΧΕΙΡΟΚΙΝΗΤΟ]:
  1. Πατήστε "Χειροκίνητο" 👆
  2. Επιλέξτε συσκευή πηγή
  3. Επιλέξτε DNS server
  4. Εισάγετε domain name

### ▶️ ΠΡΟΣΟΜΟΙΩΣΗ ΚΥΚΛΟΦΟΡΙΑΣ
• Πατήστε "Προσομοίωση" ▶️
• Το σύστημα δημιουργεί τυχαία πακέτα
• Οπτικοποίηση κυκλοφορίας
• Πατήστε ξανά για διακοπή ⏸️

## 🌐 ΠΡΟΚΑΘΟΡΙΣΜΕΝΑ ΔΙΚΤΥΑ

### 🏠 LAN ΔΙΚΤΥΟ
• Πατήστε "LAN" για τοπικό δίκτυο:
  - Router (192.168.1.1)
  - Switch
  - Υπολογιστής (192.168.1.10)
  - Server (192.168.1.100)
  - DNS Server (192.168.1.53)

### ☁️ WAN ΔΙΚΤΥΟ
• Πατήστε "WAN" για εξωτερική σύνδεση:
  - Cloud/WAN (203.0.113.1)
  - Router με WAN/LAN
  - Πλήρες LAN δίκτυο
  - Static routing για WAN

## 💾 ΑΠΟΘΗΚΕΥΣΗ & ΦΟΡΤΩΣΗ

### 💾 ΑΠΟΘΗΚΕΥΣΗ WORKSPACE
1. Πατήστε "Αποθήκευση" 💾
2. Δώστε όνομα
3. Κατεβάζει JSON αρχείο

### 📂 ΦΟΡΤΩΣΗ WORKSPACE
1. Πατήστε "Φόρτωση" 📂
2. Επιλέξτε JSON αρχείο
3. Φορτώνει συσκευές και συνδέσεις

• [ΑΥΤΟΜΑΤΗ ΑΠΟΘΗΚΕΥΣΗ]: Κάθε 30 δευτερόλεπτα

## 🔍 ΕΝΤΟΛΕΣ ΔΙΑΧΕΙΡΙΣΗΣ

### 🌐 ΔΙΑΧΕΙΡΙΣΗ DNS
1. Επιλέξτε DNS server
2. Στο πάνελ ρυθμίσεων:
   • Προσθήκη νέων records
   • Διαγραφή records
   • Εμφάνιση όλων των records

### 🚦 AUTO ROUTING
• Πατήστε "Routing" για αυτόματη ρύθμιση
• Όλες οι συσκευές χρησιμοποιούν το router ως gateway

### 🐛 DEBUG ΠΛΗΡΟΦΟΡΙΕΣ
• Πατήστε "Debug" για λεπτομερή πληροφορίες
• Εμφάνιση συνδέσεων στο console
• Έλεγχος δικτύου για προβλήματα

## 🚀 ΟΔΗΓΙΕΣ ΓΡΗΓΟΡΗΣ ΕΝΑΡΞΗΣ

1. [ΓΡΗΓΟΡΗ ΕΝΑΡΞΗ]: Πατήστε για οδηγίες
2. [DEMO]: Χρησιμοποιήστε LAN/WAN
3. [ΠΕΙΡΑΜΑΤΙΣΜΟΣ]: Προσθέστε και δοκιμάστε
4. [ΕΚΜΑΘΗΣΗ]: Παρακολουθήστε πακέτα και DNS

## ⚠️ ΣΥΜΒΟΥΛΕΣ ΑΝΤΙΜΕΤΩΠΙΣΗΣ ΠΡΟΒΛΗΜΑΤΩΝ

### 🔌 ΣΥΝΔΕΣΕΙΣ ΔΕΝ ΕΜΦΑΝΙΖΟΝΤΑΙ
• Ελέγξτε IP διευθύνσεις
• Βεβαιωθείτε ότι είναι στο ίδιο subnet
• Ελέγξτε τα gateways

### 🌐 DNS ΑΠΟΤΥΧΙΕΣ
1. Βεβαιωθείτε για ρυθμισμένο DNS server
2. Ελέγξτε IP του DNS server
3. Χρησιμοποιήστε "Χειροκίνητο DNS"

### ▶️ ΠΡΟΣΟΜΟΙΩΣΗ ΔΕΝ ΛΕΙΤΟΥΡΓΕΙ
• Βεβαιωθείτε ότι υπάρχουν συνδέσεις
• Ελέγξτε αν οι συσκευές είναι online
